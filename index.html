Hello!

This website serves as a information guide to the PB138 course project.

Introduction to project

Web application will allow users to view and edit family tree records. The developmnet of the application will be focused on using XML technologies.

Application structure

After a successful sign up, user should be able to log in to his personal interface. The user will be able to create, edit and to view his family tree records.

Family tree datas will be represented as an XML document. The XML documents will be stored in a native XML database. The backend (i.e. CRUD) will be implemented in Java. Query results will be processed and presented in the user interface on the website.

Possible enhancements:

  • Guest sessions
  • Importing / exporting family tree records

Technologies used

  • Java
  • XML
  • eXistDB (XPath, XQuery)
  • HTML
  • CSS
  • JavaScript
